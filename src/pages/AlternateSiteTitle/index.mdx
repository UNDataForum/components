# AlternateSiteTitle

The alternate site title features the `#UNDataForum` hashtag and the year.

## Props

| Prop    | Type                    | Required | Default   |
| :------ | :---------------------- | :------- | :-------- |
| `color` | [`color`](/types/color) | `false`  | `primary` |
| `year`  | `number`                | `true`   |           |

In addition `AlternateSiteTitle` accepts [margin and padding props](https://rebassjs.org/props/#margin-and-padding).
