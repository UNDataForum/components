# Badge

A simple badge.

## Props

`Badge` includes all props from type [badge](/types/badge/). It also includes the following prop:

| Prop    | Description                   | Type                     | Required |
| :------ | :---------------------------- | :----------------------- | :------- |
| `theme` | Theme applied via `Provider`. | [`theme`](/types/theme/) | `true`   |

In addition `Badge` accepts [margin and padding props](https://rebassjs.org/props/#margin-and-padding).

## Examples

### Default color

If you do not provide a `color` prop to `Badge`, it uses the default color `primary`.

<DefaultColorExample />

[Fullscreen](/Badge/default-color-example/)

```jsx
<Badge
  badge={{
    text: "Badge"
  }}
  m={2}
/>
```

### Custom color

In this example we provide a `color` prop to `Badge`.

<CustomColorExample />

[Fullscreen](/Badge/custom-color-example/)
