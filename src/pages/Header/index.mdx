# Header

Header with site title and navigation.

## Props

| Prop    | Description                                                         | Type                            | Required | Default                                |
| :------ | :------------------------------------------------------------------ | :------------------------------ | :------- | :------------------------------------- |
| `title` | Render prop that gives you full control over how title is rendered. | `func`                          | `false`  | `color => <SiteTitle color={color} />` |
| `color` | Color of font and border.                                           | `string`                        | `false`  | `primary`                              |
| `links` | Navigation links.                                                   | [`arrayOf(link)`](/types/link/) | `true`   |                                        |

In addition `Header` accepts [margin and padding props](https://rebassjs.org/props/#margin-and-padding).

## Examples

### Default color

If you do not provide a `color` prop to `Header`, it uses the default color.

[Fullscreen](/Header/default-color-example/)

```jsx
  links={[
```

```jsx
      href: "/about",
      text: "About"
    },
    {
      href: "/blog",
      text: "Blog"
    },
    {
      href: "/programme",
      text: "Programme"
    }
  ]}
/>
```

### Custom title

In this example we replace the default header title with a custom header title. Note that the `title` function has a `color` prop to ensure that the `Header`'s `color` prop is applied to the custom title as well.

<CustomTitleExample />

[Fullscreen](/Header/custom-title-example)

```jsx
<Header
  links={[
    {
      href: "/about",
      text: "About"
    },
    {
      href: "/blog",
      text: "Blog"
    },
    {
      href: "/programme",
      text: "Programme"
    }
  ]}
  title={color => <AlternateSiteTitle color={color} year={2018} />}
/>
```
