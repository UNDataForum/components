# Type badge

| Prop    | Description                            | Type                    | Required |
| :------ | :------------------------------------- | :---------------------- | :------- |
| `color` | Badge color.                           | [`color`](/types/color) | `false`  |
| `text`  | Text for badge.                        | `string`                | `true`   |
| `href`  | Link badge to this `href` if provided. | `string`                | `false`  |
