# Type theme

| Prop           | Description                                                        | Type   | Default      |
| :------------- | :----------------------------------------------------------------- | :----- | :----------- |
| `internalLink` | Render prop to render internal links. `undefined` means use `<a>`. | `func` | `undefined`  |
| `externalLink` | Render prop to render external links.                              | `func` | `NewTabLink` |
