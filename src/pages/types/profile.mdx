# Type profile

A profile describes a person.

| Prop          | Description                                                        | Type                                 | Required |
| :------------ | :----------------------------------------------------------------- | :----------------------------------- | :------- |
| `avatar`      | Render prop that gives control over how to render a profile image. | `func`                               | `true`   |
| `name`        | Profile name.                                                      | `string`                             | `false`  |
| `affiliation` | Profile affiliation.                                               | [`affiliation`](/types/affiliation/) | `false`  |
| `badges`      | Profile badges.                                                    | [`arrayOf(badge)`](/types/badge/)    | `false`  |
| `href`        | Link profile to `href` if provided.                                | `string`                             | `false`  |
