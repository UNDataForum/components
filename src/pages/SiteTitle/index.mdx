# Site Title

Default site title.

## Props

| Prop    | Type                        | Required | Default |
| :------ | :-------------------------- | :------- | :------ |
| `color` | [`colorType`](/types/color) | `false`  | primary |

Additionally, `SiteTitle` accepts all [Rebass margin and padding props](https://rebassjs.org/props/#margin-and-padding).
