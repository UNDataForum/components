`Tags` is used to render one or more tags. Unlike
[`Badges`](/components/badges), tags are always linked.

## Examples

Tags align start (left) by default:

```jsx live
<Tags
  values={[
    { tag: 'blue', path: '/tags/blue/' },
    { tag: 'red', path: '/tags/red/' },
    { tag: 'green', path: '/tags/green/' },
    { tag: 'yellow', path: '/tags/yellow/' },
  ]}
  m={2}
/>
```

You can align center:

```jsx live
<Tags
  values={[
    { tag: 'blue', path: '/tags/blue/' },
    { tag: 'red', path: '/tags/red/' },
    { tag: 'green', path: '/tags/green/' },
    { tag: 'yellow', path: '/tags/yellow/' },
  ]}
  align="center"
  m={2}
/>
```

Or align end (right):


## System props

`Tags` supports [space props](https://styled-system.com/api#space) and
[color props](https://styled-system.com/api#color) from
[Styled System](https://styled-system.com/).

## Component props

| Name    | Type                       | Default        | Required | Description                                                  |
| :------ | :------------------------- | :------------- | :------- | :----------------------------------------------------------- |
| values  | Array of shape             |                | âœ“        | Array of objects. Each object has a `tag` and a `path` prop. |
| gap     | Number                     | 2              |          | Gap between badges (from space scale).                       |
| align   | `start`, `center` or `end` | `start`        |          | Alignment (responsive).                                      |
| variant | String                     | `tags.primary` |          | Optional variant for customizations.                         |

## Variants

If you do not provide a `variant` prop, tags are rendered with the defaults from
Theme UI's
[`Badge`](https://github.com/system-ui/theme-ui/blob/master/packages/components/src/Badge.js).
You can define your own variants under any key and provide that key to
`variant`. But it is recommended to define custom variants under the `tags` key.

In this example we customize the corners of the tags using default variant
`tags.primary`:

```jsx live
<ThemeProvider
  theme={{
    tags: {
      default: {
        borderRadius: 0,
      },
      primary: {
        variant: 'tags.default',
      },
    },
  }}
>
  <Tags
    values={[
      { tag: 'blue', path: '/tags/blue/' },
      { tag: 'red', path: '/tags/red/' },
      { tag: 'green', path: '/tags/green/' },
      { tag: 'yellow', path: '/tags/yellow/' },
    ]}
    m={2}
  />
</ThemeProvider>
```

In this example we define variant `tags.secondary` and apply it to `Tags`:

```jsx live
<ThemeProvider
  theme={{
    tags: {
      secondary: {
        color: 'background',
        bg: 'secondary',
      },
    },
  }}
>
  <Tags
    values={[
      { tag: 'blue', path: '/tags/blue/' },
      { tag: 'red', path: '/tags/red/' },
      { tag: 'green', path: '/tags/green/' },
      { tag: 'yellow', path: '/tags/yellow/' },
    ]}
    variant="tags.secondary"
    m={2}
  />
</ThemeProvider>
```
