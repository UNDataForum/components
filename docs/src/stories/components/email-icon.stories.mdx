# EmailIcon

An `EmailIcon` is a square inline element. The default size is 32 pixels:

<Preview>
  <Story name="default size">
    <EmailIcon />
  </Story>
</Preview>

Use the `size` prop to set a custom size:

<Preview>
  <Story name="custom size">
    <EmailIcon size={64} />
  </Story>
</Preview>

The `size` prop can also take an array to make the icon's size responsive:

<Preview>
  <Story name="responsive size">
    <EmailIcon size={[32, 64, 128]} />
  </Story>
</Preview>

Use the `title` prop to set a title that appears when hovering over the icon:

<Preview>
  <Story name="with title">
    <EmailIcon title="Send us an e-mail" />
  </Story>
</Preview>

Since `EmailIcon` is an inline element, it uses the text color of its parent
element:

<Preview>
  <Story name="set color">
    <Box color="primary">
      <EmailIcon />
    </Box>
  </Story>
</Preview>
